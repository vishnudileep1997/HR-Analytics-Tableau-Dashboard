# HR Analytics Dashboard – Tableau Project

## Project Overview
This project is an interactive HR Analytics Dashboard developed using Tableau to analyze employee data, workforce trends, attrition patterns, and departmental performance.

The dashboard helps organizations make data-driven HR decisions through interactive visualizations and KPI tracking.

---

## Tools & Technologies
- Tableau
- Excel / CSV Dataset
- Data Visualization
- HR Analytics

---

## Key KPIs
- Employee Count
- Attrition Rate
- Average Salary
- Department-wise Analysis
- Gender Distribution
- Employee Satisfaction
- Job Role Analysis

---

## Dashboard Features
- Interactive Filters
- Attrition Analysis
- Department Performance
- Employee Demographics
- Salary Analysis
- Workforce Trends

---

## Business Insights
- Identified departments with high attrition
- Analyzed employee distribution
- Tracked workforce trends
- Compared department performance
- Evaluated employee demographics
